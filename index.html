Connect Strava

To connect your Strava account, you'll need to get an access token:

  1. Go to Strava API Settings and note your Client ID
  2. Visit this URL (replace YOUR_CLIENT_ID with your actual Client ID):
    https://www.strava.com/oauth/authorize?client_id=YOUR_CLIENT_ID&response_type=code&redirect_uri=https://guymerin.github.io/work-location-calendar/&approval_prompt=force&scope=activity:read_all
  3. Authorize the app - you'll be redirected back to your site
  4. Copy the code parameter from the URL in your browser's address bar
  5. Use a tool like Postman or HTTPie to exchange the code for a token:
    POST to https://www.strava.com/oauth/token with: client_id, client_secret, code, grant_type=authorization_code
  6. Paste the access_token from the response below

üí° Easier Alternative: Use Strava's API playground to test API calls and get a temporary token, or use browser dev tools to extract the token from a successful API call.

Note: Strava access tokens expire after 6 hours. If your workouts stop showing, simply reconnect by clicking "Connect Strava" again and pasting a new token. The app will automatically detect when your token expires and show the connect button again.
